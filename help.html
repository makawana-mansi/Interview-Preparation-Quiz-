Help & Guide

📌 How to Use

1. Type your question or subject in the search box

2. Go to the quiz page to view questions

3. Select one option for each question

4. Click the Submit button

5. View your score and results

📋 Rules

• Only one answer is allowed per question

• Answers cannot be changed after submission

• Each correct answer gives 1 mark

🔘 Buttons Info

Search → To find questions

Submit → To check your answers

Answer & Solution → To view correct answer and explanation

⚠️ Common Issues

• If no option is selected, the answer will not be checked

• Refreshing the page will reset the quiz

💡 Tips

• Read each question carefully

• Take your time before selecting an answer
